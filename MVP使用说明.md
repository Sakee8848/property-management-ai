# 🎨 物业管理AI系统 - MVP前端原型使用说明

## ✨ 什么是MVP原型？

这是一个**最小可行产品(MVP)**前端原型，特点：

- ✅ **零配置**：无需安装任何软件，无需配置后端
- ✅ **立即使用**：双击HTML文件即可在浏览器中运行
- ✅ **完整演示**：展示所有核心功能和界面
- ✅ **真实交互**：使用模拟数据，完整的用户体验
- ✅ **移动端优化**：完美适配手机屏幕

## 🚀 如何使用

### 方法1：电脑浏览器打开

1. 找到文件：`mvp-prototype.html`
2. 双击打开（会自动用默认浏览器打开）
3. 开始体验！

或者从终端/命令行打开：
```bash
open /Users/tonyyu/Documents/物业管理/mvp-prototype.html
```

### 方法2：手机浏览器打开（推荐）

**方式A：通过AirDrop**
1. 选中 `mvp-prototype.html` 文件
2. 右键 -> 共享 -> AirDrop到iPhone
3. 在iPhone上用Safari打开

**方式B：通过本地服务器**
```bash
# 在项目目录启动简单HTTP服务器
cd /Users/tonyyu/Documents/物业管理
python3 -m http.server 8888

# 然后在手机浏览器访问（确保手机和电脑在同一WiFi）：
# http://你的电脑IP:8888/mvp-prototype.html
```

### 方法3：在线预览（最简单）

直接拖拽 `mvp-prototype.html` 到浏览器窗口即可。

## 🎯 核心交互演示

### AI对话示例

**用户**: "物业费怎么缴纳？"

**AI回复**:
```
您可以通过以下方式缴纳物业费：

1. 在线缴费：点击"缴费中心"选择账单进行支付
2. 微信支付：关注物业公众号进行缴费
3. 支付宝：搜索"物业缴费"小程序
4. 现场缴费：到物业服务中心办理

物业费缴费期限为每月月底前，逾期将产生滞纳金。

📄 引用文档：物业缴费指南
```

### 支付流程演示

1. 进入"缴费中心"
2. 看到待缴费用总额
3. 点击某个账单的"立即缴费"
4. 弹出支付方式选择
5. 选择"微信支付"或"支付宝"
6. 显示"支付成功"
7. 账单状态更新为"已支付"

### 文档搜索演示

1. 进入"文档查询"
2. 点击"停车"分类标签
3. 在搜索框输入"停车"
4. 过滤显示相关文档
5. 点击文档查看详情

## 🐛 故障排查

### 问题1：页面无法打开
**解决**：确保文件完整，尝试其他浏览器
